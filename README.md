# 🚀 Portfólio Pessoal - Santiago Rocha

Este é o meu portfólio profissional, desenvolvido para apresentar meus projetos, habilidades e experiências como **Desenvolvedor Full-Stack**. O site foi construído com foco em design limpo, responsividade e performance.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica.
- **CSS3**: Estilização avançada com Grid, Flexbox e animações.
- **JavaScript (ES6+)**: Manipulação de DOM e efeitos de scroll.
- **Font Awesome**: Ícones sociais e de interface.
- **Google Fonts**: Tipografia moderna (Poppins).

## ✨ Funcionalidades

- **Scroll Suave**: Navegação fluida entre as seções.
- **Animações de Entrada**: Cards de habilidades e projetos aparecem conforme o usuário faz o scroll (Intersection Observer API).
- **Responsividade**: Layout adaptável para dispositivos móveis, tablets e desktop.
- **Integração com GitHub**: Links diretos para os repositórios dos projetos.
