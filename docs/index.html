Login com Criptografia

Este é um projeto acadêmico que demonstra um sistema completo de login e cadastro de usuários. A principal característica deste projeto é a implementação manual da Criptografia Híbrida (RSA + AES) para proteger as senhas dos usuários durante a transmissão de dados entre o cliente (navegador) e o servidor (PHP). O projeto também inclui o armazenamento seguro de senhas no banco de dados usando Hashing (password_hash).
